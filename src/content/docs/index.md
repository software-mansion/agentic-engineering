This book collects practical insights from applying agentic engineering patterns in [Software Mansion's](https://swmansion.com?utm_source=ai-workbook) own projects and in our clients' work. We are sharing it publicly so more teams can build with these methods.

:::caution

**Last revised: March 2026**
:::

Think of this document as a sidekick you keep open while you work.

## How to read this workbook

Read it in order like a short book:

1. [Getting Started](/getting-started/)
2. [Becoming Productive](/becoming-productive/)
3. [Expanding Horizons (optional)](/expanding-horizons/)

If you are short on time, use the learning paths below.

## Learning paths

- **New to agentic engineering**: Start with [Getting Started](/getting-started/), then [Becoming Productive](/becoming-productive/). Read [Expanding Horizons](/expanding-horizons/) as optional deep-dive material.

- **Already experienced**: Skim [Getting Started](/getting-started/), read [Becoming Productive](/becoming-productive/) in full, then jump to selected sections in [Expanding Horizons](/expanding-horizons/).

## Chapters

### [Getting Started](/getting-started/)

**Read this if:** Agentic engineering is new to you, or you need to set up your local environment.
**You will get:** A practical software setup, core terminology, and first workflows (first commit, first bug fix, first design implementation).
**Estimated effort:** 45-75 min (or use as a reference while building).

### [Becoming Productive](/becoming-productive/)

**Read this if:** This is the meat of the workbook. We expect everyone to study this section closely. It covers the frameworks that will actually make you a faster engineer. Think of it as a _road to 10x_.
**You will get:** Prompting and workflow patterns that make day-to-day agentic work more reliable and scalable.
**Estimated effort:** 60-90 min.

### [Expanding Horizons (optional)](/expanding-horizons/)

**Read this if:** You’re curious about the "how" and "why" behind the curtain. This is optional extra credit for deep divers.
**You will get:** Deeper mental models for context, compaction, model behavior, and MCP tradeoffs.
**Estimated effort:** 30-45 min.
