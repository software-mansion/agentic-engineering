# Software Mansion AI Workbook

This book collects practical insights from applying agentic engineering patterns in [Software Mansion's](https://swmansion.com?utm_source=ai-workbook&utm_medium=readme) own projects and in our clients' work. We’re sharing it publicly so more teams can build with these methods.

## Quickstart

```bash
bun install

# Runs locally at `http://localhost:4321`.
bun run dev

# Formats code with Prettier and runs `astro check`.
bun run lint

# Build
bun run build
bun run preview
```
